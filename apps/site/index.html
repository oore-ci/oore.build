oore

Flutter CI for Mac

A build server that runs on macOS. Point GitHub or GitLab webhooks at it, and it builds your Flutter apps using Keychain for code signing.

Building this for my own projects. Open source in case it's useful to others.

Alpha — builds, code signing, and artifacts work. Distribution is the next milestone.

What it does

  • Listens for webhooks from GitHub and GitLab
  • Runs Flutter builds (iOS, Android, macOS)
  • Signs iOS builds using macOS Keychain
  • Stores build artifacts locally
  • Runs as a launchd system service
  • CLI, REST API, and web dashboard

Install

Requires macOS.

curl -fsSL https://oore.build/install | bash

Usage

# Start daemon
~/.oore/bin/oored run --listen 127.0.0.1:8787

# Open setup window token
~/.oore/bin/oore setup token --ttl 15m

# Check health
curl http://127.0.0.1:8787/healthz

# Daemon status (example)
● running
